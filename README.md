

Task App

A simple task management app that allows users to add tasks to a list.

Features

- Add tasks to a list by entering the task name in the input field and clicking the "Add Task" button
- Tasks are displayed in a table below the input field

Technologies Used

- HTML
- CSS
- JavaScript

Getting Started

1. Open the index.html file in a web browser to use the app.
2. Enter a task name in the input field and click the "Add Task" button to add a task to the list.
3. Tasks will be displayed in the table below the input field.
